### Kết quả đạt được tuần 12:

* Hoàn thành dự án cuối khóa AWS kết hợp nhiều dịch vụ quan trọng:
  * Thiết kế kiến trúc có khả năng mở rộng với VPC, subnets, và security groups
  * Triển khai giải pháp lưu trữ với S3 và CloudFront CDN
  * Xây dựng tầng database với RDS hoặc DynamoDB
  * Tạo serverless backend với Lambda và API Gateway
  * Tích hợp monitoring với CloudWatch

* Tự thiết kế kiến trúc, triển khai, tối ưu và vận hành hệ thống thực tế:
  * Áp dụng best practices cho bảo mật (IAM, Security Groups, encryption)
  * Triển khai high availability và fault tolerance
  * Cấu hình auto-scaling để tối ưu chi phí
  * Thiết lập CI/CD pipeline cho automated deployments

* Nắm vững quy trình xây dựng ứng dụng Cloud từ đầu đến cuối:
  * Phân tích yêu cầu và thiết kế kiến trúc
  * Cung cấp và cấu hình infrastructure
  * Triển khai và kiểm thử ứng dụng
  * Monitoring, logging, và troubleshooting
  * Viết tài liệu và thuyết trình

* Hoàn thành lộ trình học AWS 12 tuần:
  * Tuần 1-5: Khám phá AWS Services (Compute, Storage, Networking, Database)
  * Tuần 6-8: Tối ưu Operations (Lambda, CloudFormation, CDK, Systems Manager)
  * Tuần 9-11: Tối ưu Security (IAM, WAF, KMS, GuardDuty, Cognito)
  * Tuần 12: Dự án cuối khóa và Tổng kết

* Sẵn sàng áp dụng kiến thức vào các dự án quy mô lớn hoặc thực tế
