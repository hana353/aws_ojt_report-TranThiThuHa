### Week 4 Achievements:

* Mastered Amazon Lightsail for simplified computing:
  * Understood differences between Lightsail and EC2
  * Created Lightsail instances with various blueprints
  * Deployed WordPress website on Lightsail
  * Configured static IP and DNS for Lightsail instance

* Learned Lightsail Containers:
  * Created container service on Lightsail
  * Deployed containerized application
  * Configured public endpoints and custom domains
  * Understood container deployment workflow

* Implemented EC2 Auto Scaling:
  * Created Launch Templates for EC2 instances
  * Configured Auto Scaling Groups (ASG)
  * Set up scaling policies (target tracking, step scaling)
  * Tested scaling based on CPU utilization
  * Understood desired, minimum, and maximum capacity

* Mastered Amazon CloudWatch for monitoring:
  * Explored default EC2 metrics (CPU, Network, Disk)
  * Created custom CloudWatch alarms
  * Set up alarm notifications via SNS
  * Created CloudWatch dashboards for visualization
  * Configured CloudWatch Logs for application logging

* Configured Amazon Route 53 for DNS:
  * Understood DNS concepts (A, CNAME, Alias records)
  * Created hosted zones for domains
  * Configured different routing policies (Simple, Weighted, Latency)
  * Integrated Route 53 with other AWS services (ALB, S3, CloudFront)
